## 安装

### 框架

https://taro-docs.jd.com/taro/docs/README/index.html

```html
脚手架：npm install -g @tarojs/cli
创建项目：taro init 项目名   
//注：如报错未找到文件命令，需修改/增加环境变量
安装依赖：npm install
启动：
npm run dev:weapp
npm run dev:h5
npm run dev:swan
npm run dev:alipay
npm run dev:tt
npm run dev:qq
打包：
npm run build:tt
```
